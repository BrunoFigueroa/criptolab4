# IMPORTANTE

**Python:** Requiere Python 3.12 o superior.  
**Librería:** PyCryptodome (no PyCrypto) por compatibilidad con Python 3.12.  

Instalación: pip install pycryptodome

**Ejecución:**  
En la terminal, ejecutar: python crypto.py

Se abrirá un formulario que pedirá:
1. Tipo de algoritmo (DES, 3DES o AES-256)  
2. Clave (key)  
3. Vector de inicialización (IV)  
4. Texto a cifrar  

El programa mostrará el texto cifrado en Base64 y luego el texto descifrado.
